---

## DEVELOPMENT

``` bash
# Install dependencies
$ npm install

# Serve with hot reload at localhost:3000
$ npm run dev
```

---

## BUILD

```bash
# Build for production and launch server
$ npm run build
$ npm start

# Generate static project
$ npm run generate
```

---

## DEPLOYMENT

```bash
# Deploy to S3
$ npm deploy
```
